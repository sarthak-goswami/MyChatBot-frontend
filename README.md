
# ChatBot Frontend

Frontend for a chatbot web application built using **React and Tailwind CSS**.  
The interface allows users to create conversations, send messages, and view chat history.

## Tech Stack

- React.js
- Tailwind CSS
- Axios
- React Router

## Features

- User Signup and Login interface
- JWT authentication integration
- Chat interface with message bubbles
- Conversation sidebar
- Create new conversations
- Profile page displaying user data
- Responsive UI

## Installation

Clone the repository

```bash
git clone https://github.com/sarthak-goswami/MyChatBot-frontend
cd MyChatBot-frontend
```

### Install dependencies
```
npm install
```

### Start the development server

```
npm run dev
```
### Application will run on: http://localhost:5173

## Backend Connection

Make sure the backend server is running.

Update API base URL if needed: http://localhost:5000/api\
